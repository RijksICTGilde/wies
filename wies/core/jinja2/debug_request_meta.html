  1. Zoek je publieke IP op (bijv. via ifconfig.me).

  3. TRUSTED_PROXY_HOPS = de positie-van-rechts van jouw IP. Staat je IP op [-1], dan is de waarde 1. Staat je IP op [-2], dan 2. Etc.
